## Roadmap

- **Front-End:**
  - [ ] Page Présentation EpiNature avec caroussel des membres de l'EpiNature en js.
  - [ ] Page Boutique.
  - [ ] Page Produit.
  - [ ] Page Blog.
  - [ ] Page Compte, Page Formulaire Infos compte, Page historique de commande.

  - [ ] Page Création(Ajout d'un compte en BDD).
